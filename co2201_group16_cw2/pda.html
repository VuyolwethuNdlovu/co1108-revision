Pushdown Automata

Brief Description of PDAs: 

Pushdown automata can deal with Non-regular grammar unlike
the previous automata. As a result, we can track the number of occurrences of elements
in strings so they can later be matched to an element which is reliant on it.
This is done using a push and pop method implementing a stack (a simple memory
device).

The stack follows a Last-In-First-Out rule when adding and
removing from a stack. For example, if the string ‘abc’ was added to a stack
one by one the order in which they would pop would be ‘cba’.

Push: Refers to adding elements to the stack

Pop: Refers to removing elements from a stack
